## 설명

표의 기한은 action deadline이다. 추출 결과의 date 필드는 문서 날짜인 2026-07-02를 사용해야 한다. 마감일은 content 안에 남아도 되지만 date 필드에 2026-07-05나 2026-07-06이 들어가면 안 된다.

## 추가 메모

이 문서는 표가 핵심이다. 본문에는 같은 내용이 길게 반복되지 않는다. 추출기는 표의 담당자와 항목을 함께 읽어야 한다. 특히 API status 응답 작업은 김민수가 담당하고, Streamlit 업로드 진행바 확인은 박지현이 담당한다.
