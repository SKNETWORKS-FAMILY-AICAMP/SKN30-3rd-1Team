4. 핵심 인사이트

  1. E0 하이브리드 검색은 유효하다. R0-both보다 precision을 크게 높이면서 높은 recall을 유지한다.
  2. E1은 일반 결정·액션보다 변경 질문의 정합성에 기여한다. 변경 recall은 .900→1.000, precision은 .422→.464다.
  3. E2의 병목은 history intent 감지다. 기대 이력 질문 5개가 모두 e2e에서 미탐됐다.
  4. Oracle 답변 품질은 아직 알 수 없다. 검색은 가능하지만 생성 지표와 답변 원문이 없다.
  5. 비번복 대조군을 확대해야 한다. E2에서 해당 faithfulness가 Modu .833→.592, CS-Bot .856→.607로 하락했다.
