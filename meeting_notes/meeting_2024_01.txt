## 2. 논의 사항

### 2.1 기술 스택 결정
- 백엔드: Python FastAPI (기존 Django에서 마이그레이션)
- 프론트엔드: React 18 + TypeScript
- AI: LangChain + GPT-4
- 데이터베이스: PostgreSQL + Redis

**결정**: FastAPI 마이그레이션 진행. 기존 API 호환성 유지 필수.

### 2.2 AI 기능 범위
- 스마트 검색: 자연어 질의 지원
- 회의록 자동 요약: 음성 → 텍스트 → 요약
- 태스크 자동 생성: 회의 내용에서 액션 아이템 추출

**결정**: 1차 버전에서는 스마트 검색만 포함. 요약 기능은 2차에서 진행.

### 2.3 리소스 및 일정
- 개발 인력: 5명 (PM 포함)
- 추가 인력 필요: AI 전문가 1명 (외부 자문)
- 예산: 5000만원

**결정**: AI 자문역 2월 초까지 섭외 완료.
